Umas das features mais modernas que existem no CSS que é as Variáveis, o que que se pode fazer com uma variável?

Imagina que queira-se repetir várias vezes uma cor que o site tenha como padrão, poderia criar uma classe .cor, mas com uma feature de variável podemos declarar assim:

{
--corPadrao: pink;
}

o nome da variável pode ser personalizado, desde que siga o exemplo acima, desde que não tenha espaço entre o texto e nenhum caractere específico, no caso do exemplo colocarei a cor como rosa e criarei outra variável que será '--widhtPadrao: 300px; Agora dentro da div{} usando o background-color utiliza-se o var() e dentro coloca-se --corPadrao, e do mesmo jeito com max-width coloca-se var(widhtPadrao).
O segredo de quando usar variável seria quando se estiver desenvolvendo um site que irá ser reutilizado a mesma cor ou dimensões varias e varias vezes, para não ficar criando classe cria-se então variáveis, tanto que o root significa a raiz, o inicio, então ele será a base em um projeto com muitas repetições.
